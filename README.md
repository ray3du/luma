## Luma Brand Colors

### Core Palette
| Name | Hex | Usage |
|---|---|---|
| Deep Indigo | `#362F78` | Primary brand color buttons, headlines, active states |
| Muted Lavender | `#B9AEDD` | Secondary accent aura motif, soft card backgrounds |
| Soft Sky Blue | `#A9C9E8` | Secondary accent breathing/calm visuals |
| Warm Cream | `#FBF6EE` | Base background (light mode) |
| White | `#FFFFFF` | Cards and surfaces |
